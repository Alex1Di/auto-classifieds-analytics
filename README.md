# auto-classifieds-analytics
End-to-end аналитическое решение на Spark, S3 и ClickHouse для анализа продуктовых метрик автоклассифайла


Учебный кейс из курса Karpov Course



Кейс: Анализ времени жизни объявлений и сегментация автомобильных брендов

Бизнес-задача
Оценить скорость продажи автомобилей на классифайде (время жизни объявлений) для оптимизации маркетинговых активностей. Требовалось определить топ-5 наиболее быстро и топ-5 наиболее медленно продаваемых марок автомобилей для запуска таргетированных пуш-рассылок. Дополнительно необходимо было проверить гипотезу бизнеса: составляют сопутствующие группы брендов от 3% до 7% от общего объема объявлений (порог рентабельности запуска кампании).

Технологический стек

Apache Spark (PySpark) & S3: Первичная обработка и денормализация снапшотов данных.
ClickHouse: Проектирование схемы хранения, оптимизация аналитических запросов.
Движки таблиц: MergeTree, AggregatingMergeTree для инкрементальной предобработки данных на лету.
Инструменты агрегации: Материализованные представления (Materialized Views), функции семейства -State/-Merge, точные квантили quantileExact.
Методология и технические решения

Расчет метрики Lifetime: Время жизни объявления определено как разница между максимальной и минимальной датами выгрузки снапшотов для каждого уникального объявления.
Борьба с выбросами и малым объемом данных: Для исключения статистического шума из анализа убраны редкие бренды (менее 10 объявлений за год). Вместо чувствительного к выбросам среднего арифметического и неустойчивой медианы было внедрено усеченное среднее (trimmed mean) на основе интервала между 0.25 и 0.75 точными квантилями.
Обработка краевых кейсов в СУБД: Реализован fallback-механизм на основе функции avgIfOrNull (с последующей заменой на медиану через coalesce), предотвращающий появление значений NaN при расчете усеченного среднего на малых выборках.
Бизнес-результаты

Сформирован точный список топ-5 наиболее ликвидных и топ-5 наименее ликвидных автомобильных брендов на платформе.
Опровергнута гипотеза маркетинга: суммарная доля целевых групп составила критически малое значение (менее 3% от общего объема объявлений). На основе полученных данных бизнес принял решение отказаться от точечных пуш-рассылок по этим брендам из-за их низкой потенциальной эффективности и пересмотреть критерии сегментации для маркетинговых кампаний.
